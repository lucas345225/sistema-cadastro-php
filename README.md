# 💻 Sistema de Cadastro de Usuários

Projeto simples de um sistema web desenvolvido com PHP e MySQL, permitindo cadastro, listagem e exclusão de usuários.

 Funcionalidades
- Cadastro de usuários
- Listagem de usuários
- Exclusão de usuários

Tecnologias utilizadas
- PHP
- MySQL
- HTML
- CSS
